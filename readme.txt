【 製作言語 】Visual Basic 6.0 ラーニングエディション

●概要
ファイルをバイト単位で編集するためのソフトです。
いわゆるバイナリエディタですが、最大の特長として、
データ表示形式が選べるという機能があります。
２進数を選べば、文字通りのバイナリエディタにもなります。
テキストエディタと似たような感覚で使えると思います。
基本的にどんなファイルでも読み書きできるため、
使い方によってはこのソフトは危険です。
必ず自己責任で使ってください。

●動作環境
Windows 95以上
要VB6ランタイム

●使い方
基本的にはテキストエディタと同じ感覚で使えます。
データ表示形式によって使えるキーが変化します。
